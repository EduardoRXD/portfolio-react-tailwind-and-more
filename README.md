Portfolio-React-TailwindCSS-andMore
==================
Made with React, TailwindCSS and more libraries coming soon...

I created this project to study more about libraries and how to make a great and modular application

Requirements:
- Git
- Node.js

----------------------------------------

Setup Instructions:

1. Clone the repository:
   git clone https://github.com/EduardoRXD/Login-Page

2. Install dependencies with `npm install`

3. Start the development server with `npm run dev`

----------------------------------------
